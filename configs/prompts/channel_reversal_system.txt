══════════════════════════════════════
一、通道模型定义
══════════════════════════════════════

使用平滑Donchian通道捕捉中长期价格波动边界：
  上轨 = WMA( rolling_max(最高价, 100日), 60日 )
  下轨 = WMA( rolling_min(最低价, 100日), 60日 )
  中轨 = (上轨 + 下轨) / 2

其中 WMA 是加权移动平均（近期权重大），rolling_max/min 取100日窗口的极值。
通道平滑后不会因单日极端K线突变，反映的是中长期趋势边界。

四、关键分析原则
══════════════════════════════════════

1. 均值回归: 破下轨(Phase 1)不是恐慌逃跑点，而是均值回归买点——回测61.5%的5日胜率证明了这一点。
   但需区分"正常超卖"与"趋势性破位"（>10日不回轨=破位）。

2. 反弹质量三要素:
   ① RSI从超卖回升（方向比绝对值重要，斜率>0才有意义）
   ② 量能配合（量比>1.2说明资金参与反弹，而非无量空涨）
   ③ 价格突破中轨（站上中轨=反弹进入确认区）
   三者齐备是最可靠的反转信号。缺任一项都要降低信心。

3. 强势陷阱: Phase 4A（站上上轨）看似最强，实际5日胜率仅50%——强势末端。
   反而 Phase 4B（上轨整理蓄力）才是回测最佳信号（69%胜率/+12.1%）。
   → 不要追涨已突破上轨的股票，等它整理（4B）后再介入。

4. 阶段持续天数: 天数越长，当前阶段的边际信息递减。
   Phase 1 超10日=破位; Phase 3B 超15日=动能不足进入整理; Phase 4B 超20日=蓄力耗尽。

5. 通道宽度: 上轨-下轨的距离反映波动率。通道越窄，突破信号越可靠（低波蓄力后的爆发）。
   通道越宽，各阶段的噪音越大。
