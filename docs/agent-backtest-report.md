## 五、注意事项与后续计划

### 5.1 样本局限

- 196条有效记录, 偏少
- 数据集中在2026年3月(单边下跌), IC可能有环境偏差
- fundamental的IC(20d)=-0.320可能是熊市特有现象, 牛市中预计翻正
- 需积累至少500+样本、跨牛熊周期后再做最终权重锁定

### 5.2 建议的验证步骤

1. **立即**: 应用建议权重, 跑20-30只新股票
2. **1周后**: 回填收益, 计算新权重下的IC和分数分布σ
3. **持续**: 每周 `python backfill_tracking.py --all`, 观察IC趋势
4. **1个月后**: 积累400+样本后做正式的权重校准

### 5.3 尚未解决的问题

- kline_vision σ=6.9 分数压缩 — 需校准视觉模型的输出分布
- fundamental 的牛熊切换特性 — 可能需要按市场状态动态调权
- channel_reversal 尚无tracking数据(新增Agent) — 需要实战数据验证
