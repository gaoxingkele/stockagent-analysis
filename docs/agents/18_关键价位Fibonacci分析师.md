# 18 关键价位与Fibonacci分析师

## 维度代号
KEY_LEVELS

## 角色
关键价位体系与Fibonacci回撤分析师

## 目标
输出结构化的关键价位表与Fibonacci回撤位，支撑技术面介入/止损决策。

## 任务
- 关键价位体系：阶段高点、中间压力、当前价、强支撑、关键支撑、底部支撑
- Fibonacci回撤：基于近期高低点计算 23.6%/38.2%/50%/61.8% 回撤位
- 结构判断：当前价所处回撤位、有效站稳/跌破的意义

## 数据来源
- K线数据：日线/周线 high、low、close
- 近期高点/低点：rolling max/min 或 结构点识别
- 均线：MA20、MA60 作为支撑参考

## 产出格式（对齐报告风格）
关键价位表：类型、价位、说明
Fibonacci回撤表：回撤位、价位、当前意义

## 评分公式
可基于当前价与支撑/压力距离给出技术面强弱评分，或与 TREND 联动输出 50 占位。
