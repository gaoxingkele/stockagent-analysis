### 1.2 内容深度差异

**参考报告突出特点：**

1. **表格化数据呈现**
   - 财务指标表：营收、净利润、毛利率、净利率，多年度 + 同比
   - 估值表：PE/PB/PS、EPS 预测、历史分位

2. **多周期趋势结构化**
   - 月线/周线/日线/60min 各级别：趋势状态（多头/空头/震荡）、结构描述

4. **Bull vs Bear 辩论**
   - 看多论点（3–5 条）
   - 看空论点（3–5 条）
   - Judge 仲裁结论与依据

5. **情景分析与可执行策略**
   - 乐观/中性/悲观：概率、触发条件、价格目标
   - 分批建仓：第一批 27 元 30%、第二批突破 29 元 30%、第三批突破 32 元 40%
   - 止损位：跌破 MA20 / MA60
   - 催化剂清单

6. **竞争格局**
   - CR3、CR5、市占率
   - 主要对手、核心优势星级

7. **风险矩阵**
   - 应收账款、流动性、毛利率、有息负债、地缘等
   - 每项标注风险等级与说明

## 三、具体改进建议

### 3.1 数据与特征层

1. **增加 Fibonacci 计算**
   - 在 `data_backend` 或 `features` 中基于日线/周线 high/low 计算波段高低点
   - 输出 23.6%、38.2%、50%、61.8% 回撤位及当前价所处区间

2. **扩充技术指标输出**
   - `kline_indicators` 已有 momentum、amplitude、shadow 等
   - 补充：RSI(6/12/24)、MACD(DIF/DEA/柱)、KDJ(K/D/J)、布林带上下轨、均线 MA5–250
   - 在报告的技术面章节以表格形式展示

3. **融资融券数据**
   - 接入 Tushare `margin_detail()` 或东方财富 API
   - 为 DERIV_MARGIN Agent 提供融资余额、融券余额、融资净买入等

### 3.2 Agent 逻辑层

1. **趋势 Agent（TREND）**
   - 输出：多周期趋势状态表（月/周/日/60min）、HH/HL/LH/LL 结构、关键价位
   - 将 `kline_indicators` 中 momentum、structure 信息转化为结构化描述

2. **技术指标 Agent（TECH）**
   - 输出：指标读数表（RSI、MACD、KDJ、布林、均线等）
   - 在 `enrich_reason` 或专门接口中让 LLM 生成表格化读数说明

3. **基本面 Agent（FUNDAMENTAL）**
   - 输出：简化财务表（营收、净利、毛利率、ROE）、估值表（PE/PB）
   - 数据来源于 `fundamentals`、`snapshot`

4. **恢复或新增 Agent**
   - **融资融券 Agent**：权重约 12%，对资金杠杆、情绪有较强解释力
   - 其余（监管、股东、经营质量、大宗套利）可按需作为可选扩展

### 3.3 报告输出层

1. **新增章节**
   - **公司概览**：代码、行业、市值、股东、业务概要（1 段+1 表）
   - **关键价位与 Fibonacci**：支撑/压力、回撤位表
   - **竞争格局**：CR3/CR5、主要对手、核心优势（数据不足时可标“待补充”）
   - **风险矩阵**：按类型划分风险等级与说明

2. **增强现有章节**
   - **技术面**：多周期趋势表 + 技术指标读数表 + 背离说明
   - **综合决策**：展示 Bull 论点、Bear 论点、Judge 仲裁结论
   - **情景分析**：乐观/中性/悲观 + 概率 + 触发条件 + 目标价
   - **仓位与止损**：分批建仓方案、止损位、主要催化剂

3. **辩论内容落地**
   - 辩论已在 orchestrator 中执行，但未写入报告
   - 可将 Bull/Bear 论点、Judge 结论写入 `final_decision.json`，再在 PDF 中展示

### 3.4 优先级建议

| 优先级 | 项目 | 工作量 | 价值 |
|--------|------|--------|------|
| P0 | Fibonacci 关键价位计算与展示 | 中 | 高 |
| P0 | 技术指标读数表（RSI/MACD/KDJ 等） | 中 | 高 |
| P0 | Bull vs Bear 辩论内容写入报告 | 低 | 高 |
| P1 | 情景分析、分批建仓/止损由 LLM 生成 | 中 | 高 |
| P1 | 公司概览结构化输出 | 低 | 中 |
| P1 | 恢复融资融券 Agent | 中 | 中 |
| P2 | 竞争格局、风险矩阵 | 中 | 中 |
| P2 | 监管/股东/经营质量等扩展 Agent | 高 | 低–中 |

## 四、小结

参考报告在**表格化数据、多周期结构、Fibonacci 价位、辩论呈现、情景与仓位策略**上明显更完整。当前实现已具备多 Agent 加权、五级决策、多模型权重与评分，但在以下方面可重点加强：

1. **数据与特征**：Fibonacci、技术指标读数、融资融券
2. **Agent 产出**：趋势/技术/基本面输出表格化、结构化
3. **报告增强 Agent**：KEY_LEVELS、COMPETITION、RISK、SCENARIO、POSITION 从占位变为实际生成
4. **辩论与决策**：将 Bull/Bear/Judge 内容显式写入 PDF，形成完整决策链路

按 P0→P1→P2 顺序推进，可逐步逼近参考报告的深度与可读性。
