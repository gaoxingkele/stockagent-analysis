## 🎯 核心调优参数

### 融合公式 (`_compose_final_score`)

```
final = 0.50·expert_consensus + 0.32·judge_adj + 0.18·risk_mapped + consensus_bonus

expert_consensus = FOMC 中位数 - 分歧惩罚(|mean-median| × 0.3, max 3)
judge_adj = judge_raw×0.65 + expert_consensus×0.15 + direction_shift + argument_bonus
risk_mapped = 20 + maxpos × 60 + rating_bonus(低+8/中0/高-10)
consensus_bonus = Judge+Trader 共识 BUY+5 / SELL-5 / 冲突-4
```

- ✅ 日级数据（A 股上市以来）

- ⚠ 无分钟级资金流
