# 多智能体股票分析报告 - 完整分析计划

## 一、报告风格与格式（参考：金风科技002202多智能体分析报告_v2.docx）

### 1.1 报告结构
1. **封面与风险提示**：标题、标的、报告日期、分析框架、风险提示
2. **多智能体分析框架说明**：智能体模块表（分析维度、核心职能）
3. **公司概览**：股票代码、上市时间、公司全称、注册地、行业、市值、业务、股东背景、行业地位
4. **基本面分析**：核心财务数据表、估值分析表、结论
5. **技术面分析**：趋势与多周期结构、关键价位体系、Fibonacci回撤、均线系统、技术指标量化表、背离识别、A股特色指标、综合评分与结论
6. **情绪面分析**：情绪因素表、结论
7. **宏观政策分析**：政策/宏观因素表、结论
8. **行业竞争格局**：竞争对手矩阵、结论
9. **风险管理**：风险类型×等级×详细说明表、结论
10. **综合决策**：17智能体框架说明、核心9维度加权评分、Bull vs Bear 辩论、Judge 仲裁、各智能体评分快照、情景分析、最终投资建议（评级、目标价、分批建仓、止损位、催化剂）
11. **免责声明**

### 1.2 格式特点
- 大量使用**表格**呈现结构化数据
- 各章节有**结论**段落
- 技术面含**多级别**（月线/周线/日线）趋势状态表，不含分钟级
- 最终建议含**情景分析**（乐观/中性/悲观）与**可执行策略**（分批建仓、止损）

## 五、技术指标与研判级别说明

技术指标、量价、结构分析**仅采用日线、周线、月线**，不含30min/1h/2h等分钟级。常用技术指标包括：

| 指标 | 参数/说明 |
|------|----------|
| MACD | 12/26/9，DIF/DEA/柱状线 |
| RSI | 14周期 |
| KDJ | 9/3/3 |
| 布林带 | 20周期、2倍标准差 |
| StochRSI | 14周期 |
| 波动率 | 20周期标准差 |
| 多条均线 | MA5/MA20/MA60/MA120等 |
| 3～5根K线组合 | 三连阳、三连阴、两阳夹一阴、早晨之星等 |
| 长期趋势线 | 20周期线性回归斜率 |

报告中将标明各结论所在研判级别（日线/周线/月线）。
