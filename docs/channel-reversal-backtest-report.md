## 一、指标定义

### 通道构建

```
上轨 = WMA(60) of rolling_max(high, 100)
下轨 = WMA(60) of rolling_min(low, 100)
中轨 = (上轨 + 下轨) / 2
```

以100根K线的最高价/最低价分别做WMA60平滑，形成类布林带的动态通道。上轨���中轨之间为上半区（多头区），中轨与下轨之间为下半区（空头区）。

### 8阶���状态机

| 阶段 | 含义 | 触发条件 |
|---|---|---|
| 0U | 上半区(多头) | close > 中轨 |
| 0D | 下半区(空头) | close < 中轨 |
| 1 | 破下轨(超卖) | close < 下轨 |
| 2 | 回到轨道(反弹启动) | 从Phase1回到下轨上方 |
| 3A | 弱反弹 | Phase2中量弱+RSI低迷超5日仍未过中轨 |
| 3B | 过中轨(看多确认) | 从Phase2突破中轨 |
| 4A | 有效反转 | 连续3日站上上轨 |
| 4B | 上轨整理(蓄力) | 触及上轨但未能连续3日站稳 |

### 辅助因子

- RSI(14): 超卖/超买判定 + 5日斜率（方向确认）
- 量比(vol/vol_ma20): 放量确认信号有效性

## 三、核心发现

### 3.1 高分区信号可靠

75分以上的信号（4B上轨���理、3B过中轨满配）具有明显的预测能力：

- **75-85分**: 5日胜率59.7%，20日均涨+9.59%，20日胜率65.6%
- **85-95分**: 5日胜率61.8%，20日均涨+13.47%，20日胜率67.8%

这意味着通道指标的高分是有效的做多信号。

### 3.2 中间区是死区

45-65分区间包含��47,897个样本（67%），5日胜率仅46-47%，20日均涨仅+2.3%，表现最差。这个区间对应0U(上半区)和Phase2(回到轨道)，属于方向不明的震荡状态，不应产生交易信号。

### 3.3 低分区有均值回归效应（U形关系）

25-35分（破下轨/弱反弹）的20日胜率高达68.4%，均涨+4.93%。这是A股典型的超卖反弹特征：

- **Phase 1(破下轨)**: 61.5%胜率 — ���破通道后短期反弹概率高
- **Phase 3A(弱反弹)**: 56.3%胜率 — 被判定为"��"的反弹实际仍��正收益

这导致评分与收益呈**U形关系**而非线性关系，IC≈0（线性相关捕捉不到U形）。

### 3.4 4A(有效反转)名不副实

4A阶段（连续3日站上上轨，评分原92→校准后69）5日胜率仅50%，说明连续站上上轨后短期已经超买，面临回调压力。但20日+5.71%表明中期仍正，不是反转失败，而是需要消化短期获利盘。

### 3.5 4B(上轨整理)是最���信号

4B阶段（触及上轨但未连续突破，进入整理）各项指标全面最优：
