# Agent 评分逻辑详解

> 基于 `configs/agents_v2/` 的 12 个启用 Agent，代码位于 `src/stockagent_analysis/agents.py`

## 通用框架

每个 Agent 的评分流程：
1. **本地策略** (`_simple_policy`) — 基于量化数据计算基础评分（0-100），基准 50 分
2. **LLM增强** — 将 `_build_data_context` 构建的数据摘要发送给多个 LLM Provider 进行研判
3. **融合输出** — 本地分 + LLM分 按 `llm_base_weight` 配置融合

评分含义：>70 买入信号，50-70 持有/观望，<50 卖出信号

## 1. trend_momentum_agent — 趋势动量分析

| 字段 | 值 |
|---|---|
| 权重 | 0.15 |
| dim_code | TREND_MOMENTUM |
| 必选 | 是 |
| LLM权重 | 0.35 |

### 本地评分公式 `_score_trend_momentum`

基准 50 分，叠加以下因子：

| 因子 | 逻辑 | 分值范围 |
|---|---|---|
| **MA排列** | 统计 MA5/10/20/60/120 相邻对的大小关系。全多头排列(短>长) +20，全空头 -20，线性插值 | [-20, +20] |
| **趋势斜率** | 日线 `trend_slope_pct x 80`，截断 | [-15, +15] |
| **趋势线突破** | 下降趋势线被突破且确认 +12（未确认 +5），上升趋势线破位 -12/-5 | [-12, +12] |
| **乖离率惩罚** | MA5乖离 >8% 扣25分，>5% 扣15，>3% 扣5；乖离>8% 时强制封顶35分 | [-25, 0] |
| **ADX修正** | ADX>40 强趋势方向加8（反向-8）；ADX>25 +/-4；ADX<15 不调整 | [-8, +8] |
| **动量** | 20日动量 x 0.3 | 连续变量 |

### LLM上下文

通用基础数据（行情、指标、K线概览）。

## 2. capital_liquidity_agent — 资金流动性

| 字段 | 值 |
|---|---|
| 权重 | 0.12 |
| dim_code | CAPITAL_LIQUIDITY |
| 必选 | 是 |
| LLM权重 | 0.35 |

### 本地评分公式 `_score_capital_liquidity`

基准 50 分：

| 因子 | 逻辑 | 分值范围 |
|---|---|---|
| **OBV趋势** | OBV上升 +8，下降 -8，持平 0 | [-8, +8] |
| **量比** | `(vr-1.0) x 18`，截断 | [-15, +15] |
| **换手率** | >15% 过度换手 -6；8-15% -2；<0.5% 流动性不足 -4；1-5% 健康 +3 | [-6, +3] |
| **量价信号** | 放量突破 +8、缩量回踩 +6、底部放量 +4、天量滞涨 -8（可叠加） | [-8, +18] |
| **涨跌幅** | pct x 0.2 | 连续变量 |

### LLM上下文

量比、换手率、量价信号标签、OBV趋势、筹码分布（获利比例、集中度）。

## 3. fundamental_agent — 基本面分析

| 字段 | 值 |
|---|---|
| 权重 | 0.10 |
| dim_code | FUNDAMENTAL |
| 必选 | 是 |
| LLM权重 | 0.45 |

### 本地评分公式 `_score_fundamental`

基准 50 分，五大模块叠加：

**PE分级：**
- PE<15 +12，15-25 +5，40-60 -4，60-100 -8，>100 -10，<0(亏损) -10

**PB分级：**
- PB<1 +8，1-2 +4，8-15 -3，15-50 -5，>50 -8

**PEG估值（PE / 净利增速）：**
- PEG<0.5 +8，0.5-1 +4，2-3 -3，>3 -6

**F-Score简化（`_calc_fundamental_extra`）：**

| 子因子 | 逻辑 |
|---|---|
| ROE | >20% +8，>15% +5，>10% +3，<0% -6 |
| 成长性 | 营收+净利双>30% +10，双>15% +5，营收<-10%或净利<-20% -8 |
| 负债率 | >80% -6，>70% -4，>60% -2，<30% +3 |
| 流动比率 | <1.0 -4，<1.5 -2，>2.5 +2 |
| 速动比率 | <0.8 -3，<1.0 -1，>1.5 +2 |
| 同业PE对比 | PE/行业均PE <0.6 +6，<0.85 +3，>1.5 -5，>1.2 -2 |

**筹码分布：**
- 获利比 >80% +6，>60% +3，<30% -6
- 套牢比 >60% -5
- 健康评分偏差 (health-50) x 0.08

### LLM上下文

通用数据 + ROE、营收增速、净利增速、资产负债率、流动比率、速动比率等基本面字段。

## 5. chanlun_agent — 缠论买卖点

| 字段 | 值 |
|---|---|
| 权重 | 0.15 |
| dim_code | CHANLUN |
| 必选 | 否 |
| LLM权重 | 0.20 |

### 本地评分公式 `_score_multi_tf`

通用多周期加权评分：

- 从日/周/月线的 `kline_indicators[tf].chanlun.chanlun_score` 取值
- 权重分配：**日线 50% + 周线 35% + 月线 15%**
- scale=1.0
- 加权汇总后截断到 [10, 90]

### LLM上下文 (`ChanlunAgent._build_data_context`)

每个周期提供：
- 最后一个中枢（价格区间 + 笔数）
- 最近 3 笔方向和价格
- 买卖点信号列表
- 缠论评分

要求 LLM 判断：当前缠论阶段（上涨/下跌/中枢震荡）、买卖点信号、中枢指引意义。

## 7. pattern_agent — K线形态综合

| 字段 | 值 |
|---|---|
| 权重 | 0.08 |
| dim_code | PATTERN |
| 必选 | 否 |
| LLM权重 | 0.25 |

### 本地评分公式 `_score_pattern`

基准 50 分，三模块合并：

**(a) K线形态：**
- 多周期（日50%/周35%/月15%）统计 bullish/bearish 形态
- 取最高置信度 + 其余 x0.25，净值截断 +/-15

**(b) 图表形态：**
- 多周期 `chart_pattern_score x 0.5` 加权

**(c) 顶底结构：**
- 上影线比 >40% 且动量<0 → 顶部信号 +8；下影线比 >40% 且动量>0 → 底部信号 +8
- 底部抬升评分，顶部压制评分，x1.2

**连续性修正：**
- 连续3阳 + 体量递增 +3；连续3阴 + 递增 -3
- 连续更高高点 +2；连续更低低点 -2

### LLM上下文 (`PatternAgent._build_data_context`)

每周期的 K 线形态列表（名称+方向+置信度+位置百分比）、图表形态列表、顶底结构综合分、连续性统计。

## 9. kline_vision_agent — K线视觉分析

| 字段 | 值 |
|---|---|
| 权重 | 0.06 |
| dim_code | KLINE_VISION |
| 必选 | 否 |
| LLM权重 | 0.50 |

### 执行流程（与其他 Agent 完全不同）

**有图 + 有视觉模型时：**
1. 加载日线K线图（PNG，含MA/布林/MACD/RSI/KDJ/趋势线）
2. 发送给支持视觉的 LLM（gemini/grok/claude 等）
3. 多模型投票，取平均分

**无图或无视觉模型时**，降级为 `_score_kline_vision_fallback` 文本评分：

| 因子 | 逻辑 | 分值范围 |
|---|---|---|
| RSI | >75 -8，<25 +8 | [-8, +8] |
| 趋势斜率 | slope x 60，截断 | [-10, +10] |
| 动量 | +0.3 x mom | 连续变量 |
| 量比 | (vr-1) x 8 | 连续变量 |
| 波动 | -0.15 x vol | 连续变量 |

### 视觉Prompt

要求 LLM 分析：趋势方向、支撑/压力位、MACD/RSI/KDJ信号、量价配合、趋势线突破。

## 10. resonance_agent — 多周期共振

| 字段 | 值 |
|---|---|
| 权重 | 0.06 |
| dim_code | RESONANCE |
| 必选 | 否 |
| LLM权重 | 0.25 |

### 本地评分公式 `_score_resonance`

两部分组成：

**(a) 多周期共振：**

判断日/周/月各自方向 — 斜率>0.05且动量>0为bullish，斜率<-0.05且动量<0为bearish，否则neutral：

| 组合 | 评分 |
|---|---|
| 三周期全部 bullish | 78 |
| 两 bullish + 无 bearish | 66 |
| 有多有空（矛盾） | 45 |
| 两 bearish + 无 bullish | 34 |
| 三周期全部 bearish | 22 |

**(b) 相对强弱调整：**

| 因子 | 逻辑 |
|---|---|
| vs沪深300超额收益20日 | >15% +10，>8% +6，>3% +3，<-3% -3，<-8% -6，<-15% -10 |
| RS趋势 | up +4，down -4 |
| 多层共振 | vs大盘/vs行业/vs龙头三层全up +4，全down -4 |

### LLM上下文

日/周/月趋势斜率+动量+RSI，相对强弱数据（vs沪深300超额收益、RS趋势、vs行业）。

## 12. channel_reversal_agent — 动态通道反弹 (新增)

| 字段 | 值 |
|---|---|
| 权重 | 0.10 |
| dim_code | CHANNEL_REVERSAL |
| 必选 | 否 |
| LLM权重 | 0.25 |

### 通道构建

```
上轨 = WMA(60) of rolling_max(high, 100)
下轨 = WMA(60) of rolling_min(low, 100)
中轨 = (上轨 + 下轨) / 2
```

### 8阶段状态机

| 阶段 | 含义 | 基础分 | 触发条件 |
|---|---|---|---|
| **0U** | 上半区(多头) | 55 | close > 中轨 |
| **0D** | 下半区(空头) | 35 | close < 中轨 |
| **1** | 破下轨(超卖) | 25 (+5 if RSI<20) | close < 下轨 |
| **2** | 回到轨道(反弹启动) | 35 (+10 if RSI回升+放量) | 从Phase1回到下轨上方 |
| **3A** | 弱反弹(大概率破新低) | 15 | Phase2中量弱+RSI低迷超5日仍未过中轨 |
| **3B** | 过中轨(看多确认) | 60 (+10~15 if RSI>40+放量) | 从Phase2突破中轨 |
| **4A** | 有效反转(强突破) | 90 (+5 RSI, +5 放量) | 连续3日站上上轨+放量 |
| **4B** | 上轨整理(蓄力) | 60 (-5 if 整理>10日) | 触及上轨但未能连续3日站稳 |

### 辅助因子

- **RSI(14)**：超卖区反弹加分（Phase1 RSI<20 +5），Phase3B RSI>40且上升 +10
- **量比(vol/vol_ma20)**：放量>1.2 在 Phase2/3B 额外 +5，Phase4A 放量>1.5 +5
- **三因子齐备**：Phase3B 中 RSI>40 + RSI斜率>1 + 量比>1.2 → 强制至少75分

### 超时机制

| 阶段 | 超时天数 | 退化目标 |
|---|---|---|
| Phase 1 (破下轨) | >10日 | 退化为 0D |
| Phase 2 (反弹中) | >20日 | 退化为 3A (弱反弹) |
| Phase 3B (过中轨) | >15日 | 退化为 4B (上轨整理) |
| Phase 4B (上轨整理) | >20日 | 退化为 0U |

### LLM上下文

- 上/中/下轨数值
- 当前阶段 + 持续天数
- 近20根K线的阶段变迁路径（如 `破下轨(3日) → 回到轨道(5日) → 过中轨(2日)`）
- 价格在通道中的百分位（0%=下轨，100%=上轨）
- 距各轨距离百分比
